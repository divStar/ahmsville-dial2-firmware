Detailed Description

Provides basic functions, that are used by every subclass of this class.

This class should be used as an abstract class, that provides a task type and default functions, which every subclass will override.

Since the functions onSetup and onCallback are virtual, yet defined, subclasses should override them if they need to be invoked during setup() or loop() of the main.h file.

virtual void DialTask::onCallback ( )
inlinevirtual

Function, which usually runs in the loop() function of the main.h file.

The purpose of this function is to be called repeatedly each time a task is executed.

Implementing this function is optional since some tasks are interrupt-driven and do not run within the loop() function.

Reimplemented in CapacitativeTouchTask, MessagesCleanerTask, HapticTask, InputProcessorTask, KnobTask, LedTask, and SpaceNavigatorTask.

◆ onSetup()

virtual void DialTask::onSetup ( )
inlinevirtual

Function, which usually runs in the setup() function of the main.h file.

The purpose of this function is to set up whatever task is supposed to run.

Implementing this function is optional since some tasks do not need to be set up first.

Reimplemented in CapacitativeTouchTask, KnobTask, LedTask, MacroKeyTask, and SpaceNavigatorTask.
