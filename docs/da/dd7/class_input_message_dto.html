Detailed Description

DTO (data transfer object) used to transport a message from the Serial port to a sensor task.

This class describes a DTO (data transfer object), which is used to transport a message from the Serial port to a sensor task. It also provides a function to make it possible to determine whether this object is still valid or subject to be pruned by the MessagesCleanerTask.
