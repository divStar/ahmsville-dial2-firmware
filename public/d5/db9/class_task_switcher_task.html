Detailed Description

Task to handle switching individual tasks on and off.

This class is a ISchedulableDialTask, that handles the switching on and off of individual tasks.

This task is useful for controlling individual tasks.

It will not help with multiple versions of the AhmsVille Dial 2 since the onSetup() functions of individual tasks are executed regardless.
